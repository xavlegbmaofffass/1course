- дайте определение функции;
это именованный блок кода, который выполняет определенную задачу

- чем отличаются функция и процедура;
Функция возвращает значение, процедура не возвращает значение

- поясните разницу передачи параметров по значению и по ссылке;
По значению:
Функция получает копию аргумента, изменения параметра внутри функции не влияют на оригинальную переменную.
По ссылке:
Функция работает с оригинальной переменной, изменения параметра внутри функции отражаются на переданной переменной.

- перечислите известные вам соглашения о вызове функции, поясните особенности каждого соглашения;
    // cdecl: Аргументы передаются справа налево, стек очищается функцией вызывающей
    // stdcall: То же самое что и cdecl только стек очищается самой вызываемой функцией
    // fastcall: Аргументы старается положить в регистры, для ускорения быстродействия, стек очищает вызываемой

- дайте определение соглашению вызовах __cdecl;
 соглашение вызова, при котором аргументы передаются справа налево, стек очищается функцией вызывающей

- дайте определение соглашению вызовах __stdcall;
соглашение вызова, при котором аргументы так же передаются слева направо, но стек очищается вызываемой функцией

- дайте определение соглашению вызовах _fastcall
соглашение вызова, который аргументы старается положить в регистры, стек очищает вызываемой функцией
