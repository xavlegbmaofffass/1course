 дайте определение статической библиотеки;
это файл с объектными модулями, подключаемыми к программе на этапе компоновки

 назначение статической библиотеки.
Библиотеки преобразуются компилятором в объектные файлы. Затем компоновщик соединяет объектные файлы библиотек и объектные файлы программы в один исполняемый файл.

 чем отличается проект Visual Studio «приложение» от проекта «статическая библиотека»?
Приложение это полноценный исполняемый файл, который содержит необходимые ресурсы для выполнения программы.
Статическая библиотека это набор объектных файлов, которые могут быть использованы в других проектах.

 на каком этапе создания exe-модуля используются файлы статической библиотеки?
на этапе линковки

 требуется ли наличие файла с расширением lib при выполнении exe-модуля?
не требуется
